Инструменты

Kali Linux

Операционная система для тестирования безопасности. Я использую её как платформу для обучения и практики в виртуальных лабораториях.

OSINT

Сбор информации из открытых источников: соцсети, публичные базы, поисковые системы. Это важно для исследования угроз и расследований.

Metasploit

Фреймворк для тестирования безопасности. Я изучаю его в учебных целях, чтобы понимать, как работают эксплойты и как защищаться от них.

Burp Suite

Инструмент для тестирования веб-приложений (в т.ч. перехват запросов). Использую для анализа безопасности сайтов в безопасной среде.

Nmap

Сканер сети для анализа открытых портов и служб. Применяется в легитимных проверках безопасности.

Python

Я пишу скрипты для автоматизации задач, анализа данных и создания небольших инструментов для обучения.
